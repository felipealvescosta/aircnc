## Getting Started

To get a local copy up and running follow these simple steps.


### Installation
 
1. Clone the repo
```sh
git clone https://github.com/felipealvescosta/omnistack9.git
```
Run Server
1. Enter the server folder
```sh
cd server/ 
yarn install 
yarn dev
```
Run Front
2. Enter the  frontend folder
```sh
cd frontend
yarn install
yarn start
```
Run Mobile App
3. Enter the  mobile folder
```sh
yarn install
```
* 1 - Run iOS
```sh
  yarn ios
```
* 2 - Run Android
```sh
  yarn android
```
